# Web Portfolio
An personal web portfolio written in javascript with ReactJS.

###  Used libraries
 - ReactJS
 - TailwindCSS
 - FontAwesome React

## Scripts

### Install dependencies
 `yarn install `
### Start local project
`yarn start`
### Build project (Static files)
`yarn build`
